# Results

This folder contains the **outputs produced by the replication pipeline** of the project  
*“War and Institutional Change: The Case of the Gustav Line”*.

The analysis is organized around a **Geographic Regression Discontinuity Design (Geo-RDD)**, where the running variable is the **signed distance to the Gustav Line** (cutoff at 0). Municipalities north of the line are treated, and municipalities south of the line are controls. Tables and figures summarize descriptive patterns, validity checks, and estimated treatment effects.

All results in this folder are generated automatically by the scripts in `scripts/`.  
To reproduce everything, run:

- `scripts/00_run_all.R`

## Folder structure

- `tables/` — machine-readable outputs in `.csv`
- `figures/` — graphical outputs in `.png`

---

## Tables

All tables are saved as `.csv` files for transparency and easy reuse.

- `tables/table1_summary_stats.csv`  
  Summary statistics for the full Italy sample and for municipalities within ±100 km of the Gustav Line.  
  *(generated by `scripts/12_table_1_summary_stats.R`)*

- `tables/table2_balance_checks.csv`  
  Covariate balance/continuity checks around the cutoff (multiple bandwidths), with robust and Conley standard errors.  
  *(generated by `scripts/13_table_2_balance_checks.R`)*

- `tables/table3_main_results.csv`  
  Main Geo-RDD estimates across bandwidths (100/75/50 km), with and without controls, using robust and Conley standard errors.  
  *(generated by `scripts/14_table_3_main_results.R`)*

- `tables/table4_rdrobust_main_results.csv`  
  Local polynomial RD estimates computed with `rdrobust` using fixed bandwidths (100/75/50 km), with and without covariates.  
  *(generated by `scripts/15_table_4_rdrobust_main_results.R`)*

- `tables/table5_rdrobust_datadriven.csv`  
  `rdrobust` estimates using data-driven bandwidth selection (e.g., MSE/CER), with and without covariates.  
  *(generated by `scripts/16_table_5_rdrobust_datadriven.R`)*

- `tables/table6_rddensity_mccrary.csv`  
  McCrary density test results (running variable manipulation/sorting check) computed using `rddensity`.  
  *(generated by `scripts/17_table_6_rddensity_mccrary.R`)*

## Figures

All figures are saved as `.png`.

- `figures/figure1_map_italy_occupation.png`  
  Map of Italian municipalities colored by occupation duration, with the Gustav Line overlay.  
  *(generated by `scripts/18_figure_map_italy_occupation.R`)*

- `figures/figure2_map_within100km_republic.png`  
  Map of municipalities within ±100 km of the Gustav Line colored by 1946 Republic vote share (missing outcomes shown as “Excluded”).  
  *(generated by `scripts/19_figure_2_map_within100km_republic.R`)*

- `figures/figure3_rd_plot_main.png`  
  Main Geo-RDD visualization: binned scatter of Republic vote share vs signed distance, with fitted curves and confidence intervals.  
  *(generated by `scripts/20_figure_3_rd_plot.R`)*

- `figures/figure4_rdrobust_plot.png`  
  Standard RD plot produced with `rdrobust::rdplot`.  
  *(generated by `scripts/21_figure_4_rdrobust_plot.R`)*

- `figures/figure5_mccrary_plot.png`  
  McCrary density test plot at the cutoff produced using `rddensity`.  
  *(generated by `scripts/22_figure_5_mccrary_plot.R`)*
